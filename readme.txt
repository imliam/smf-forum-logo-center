[center][size=12pt][u][b]Compatibility[/b][/u][/size][/center]

The "[i]Forum Logo Center[/i]" is compatible with both SMF [b]1.1.X[/b] aswell as [b]2.0[/b].They each edit different parts of the file(s), however still work the same.

[center][size=12pt][u][b]Features[/b][/u][/size][/center]

The Forum Logo Center Modification Center's the SMF Logo on your Forum. It only works with "[i]Core[/i]" SMF Theme, and varients of it, however it might work with others.

[center][size=12pt][u][b]Installation[/b][/u][/size][/center]

Install this modification through the package manager. It only edits one theme file in a minor place, so won't go wrong unless you've edited the SMF Logo.
